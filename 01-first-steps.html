👋 Go nedir? What is Go?

⬡ Google · 2009 · açık kaynak Google · 2009 · open source

Go = hızlı, basit, derlenir. Go = fast, simple, compiled.

Sunucular, CLI araçları, ağ servisleri için ideal.
Az sözdizimi. Az gizem. Çok iş. Built for servers, CLI tools, network services.
Small syntax. Few surprises. Gets things done.

3/12

📄 İlk dosya: main.go First file: main.go 

package main

import "fmt"

func main() {
    fmt.Println("Merhaba, Go!")
}

main.go adlı dosya yarat. Bu kodu yapıştır. Create a file called main.go. Paste this code.

5 satır. Tüm Go programları böyle başlar.
Şimdi her satıra tek tek bakalım. 5 lines. Every Go program starts like this.
Let's break it down line by line.

Hangi parça ne işe yarıyor? What does each part do?

4/12

🏗️ Anatomi Anatomy

package main→ çalıştırılabilir program
import "fmt"→ kullanılacak paket
func main()→ programın başladığı yer
fmt.Println(...)→ ekrana yazdır
package main→ executable program
import "fmt"→ package to use
func main()→ where the program starts
fmt.Println(...)→ print to screen

Sıra önemli: package → import → func. Order matters: package → import → func.

Her Go dosyası bu sırayla başlar. Ezberleme — kullandıkça oturacak. Every Go file starts this way. Don't memorize — it'll click as you write.

Çalıştıralım! Let's run it!

5/12

▶️ Çalıştır: go run Run it: go run 

$ go run main.go
Merhaba, Go!

go run main.go — derler ve çalıştırır. go run main.go — compile and run.

Tek komutla. Geliştirme sırasında en sık yazacağın komut.
Binary üretmez. Sadece çalıştırır. One command. The one you'll type most while learning.
Doesn't make a binary. Just runs.

Yazdırma araçları? Print tools?

6/12

📤 fmt.Println — yazdır fmt.Println — print 

fmt.Println("Merhaba")
fmt.Println("Bugün", "Pazartesi")

// Merhaba
// Bugün Pazartesi

Println yazdırır. Sona satır sonu ekler. Println prints. Adds a newline at the end.

Birden çok argüman ver — aralarına otomatik boşluk koyar. Pass multiple arguments — it adds spaces between them.

Daha kontrollü yazdırma? More control?

7/12

🎨 fmt.Printf — şablonla fmt.Printf — with a template 

isim := "Ada"
yas := 12

fmt.Printf("%s, %d yaşında.\n", isim, yas)

// Ada, 12 yaşında.

Printf yer tutucularla yazar. \n satır sonu. Printf uses placeholders. \n is newline.

%s string · %d sayı · %v her şey · %f ondalık
Println'in aksine satır sonunu otomatik koymaz. %s string · %d integer · %v any · %f float
Unlike Println, doesn't add a newline by itself.

Koda not eklemek? Adding notes to code?

8/12

💬 Yorum satırı Comments 

// Tek satır yorum / single-line comment

/*
   Çok satırlı yorum
   Multi-line comment
*/

fmt.Println("hi")  // satır sonu yorumu

// tek satır · /* ... */ çok satır. // single line · /* ... */ multi-line.

Yorumlar derlenmez. Kendinden ya da ekibinden sonra okuyacaklara not. Comments don't run. They're notes for you and your team later.

İlk derleme hatan… Your first compile error…

9/12

🚫 Tuzak: kullanılmayan import Trap: unused import 

package main

import "fmt"
import "strings"      // ✗ kullanılmıyor

func main() {
    fmt.Println("hi")
}

// ./main.go:4: imported and not used: "strings"

Kullanılmayan import → derleme hatası. Unused import → compile error.

Aynı kural değişkenler için de geçerli.
Go disiplinli — yazıp kullanmadığın hiçbir şeye izin vermez. Same rule applies to variables.
Go is strict — anything declared must be used.

Birden çok dosya gerekirse? What about multiple files?

10/12

📦 Modül oluştur Create a module 

$ mkdir merhaba && cd merhaba
$ go mod init merhaba
$ # main.go yaz, sonra:
$ go run .

go mod init projeni başlatır. go mod init starts your project.

Bir go.mod dosyası oluşur — projenin sınırını çizer.
Birden çok dosya, dış paket veya paylaşım için şart. Creates a go.mod file — defines your project boundary.
Required for multiple files, third-party packages, or sharing.

Programı nasıl paylaşırsın? How do you share your program?

11/12

🔨 go run vs go build go run vs go build 

$ go run main.go        # derle + çalıştır

$ go build              # derle, dosya üret
$ ls
main.go   merhaba       # ← çalıştırılabilir

$ ./merhaba
Merhaba, Go!

run = öğren · build = paylaş. run = learn · build = ship.

go build tek dosyalı bir program üretir.
Karşı tarafta Go kurulu olmasa da çalışır. go build produces a single executable file.
Runs even if Go isn't installed on the target machine.
