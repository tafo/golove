# GoLove

> Go programlama dilini kart kart öğren — Learn Go, one card at a time.

GoLove is a card-based learning resource for the **Go programming language**, designed for absolute beginners. Each chapter contains 10–14 cards. Each card teaches a single concept in 5–10 seconds. Browsable like a Reels feed: scroll, swipe, or press <kbd>↓</kbd> / <kbd>Space</kbd> to advance.

- 🇹🇷 **Türkçe** ve 🇬🇧 **English** — sağ üstteki düğmeden anında değiş
- 📱 Mobile-first, masaüstünde de iyi görünür
- ⌨️ Klavye navigasyonu: <kbd>↓</kbd> <kbd>↑</kbd> <kbd>Space</kbd> <kbd>Home</kbd> <kbd>End</kbd>
- 🚀 Sıfır build adımı — saf HTML + CSS + JS
